HTML-perusopas

Tässä on HTML-perusopas, joka esittelee tärkeimmät HTML-tagit.

<html>

Kaikki HTML-sisältö laitetaan <html>-tagin sisään.

<head>

<head>-osioon laitetaan mm. otsikko ja tyylit.

<body>

Tässä osassa näkyy kaikki sivun varsinainen sisältö.

<p>

Tämä on kappale. Kappale tehdään <p>-tagilla.

CSS:n käyttö

CSS määrittelee, miltä HTML-elementit näyttävät. Sitä käytetään erottamaan sivun ulkoasu sen rakenteesta. CSS:llä voi säätää esimerkiksi värejä, fontteja, marginaaleja ja asettelua.

Esimerkki CSS-säännöstä:

p {
        color: blue;
        font-size: 16px;
      }

Tämä esimerkki muuttaa kaikki <p>-tagin kappaleet sinisiksi ja asettaa fontin kooksi 16 pikseliä.

Lisäksi sen sijaan, että kaikki tyylit tehtäisiin itse, voidaan käyttää valmiita kirjastoja kuten Bootstrap. Bootstrap tarjoaa valmiiksi tyyliteltyjä painikkeita, asetteluja, lomakkeita ja paljon muuta.
